2. Click Menu 'Import Gmail Contacts' -> 'Allow permissions'
3. Allow permissions:
- Click continue, 
- Select your account, 
(sometimes: 
  - click advanced, 
  - scroll down, 
  - unsafe go to contacts)
- Click allow"
4. Menu 'Import Gmail Contacts' -> 'Import Contacts to Sheet'
